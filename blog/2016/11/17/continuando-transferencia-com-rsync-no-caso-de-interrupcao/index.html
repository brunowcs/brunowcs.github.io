Continuando Transferência Com RSYNC No Caso De Interrupção

Precisei realizar uma transferência de uma VM do XenServer com 300G de um ponto de montagem NFS para um outro local remoto e para evitar problemas de interrupção, utilizei o comando abaixo que me permiti continuar a transferência em caso de problemas.

# rsync -vrlPtz --append /mnt/web.xva /storage/export/
    sending incremental file list
    web.xva     4641783808   1%   6.89MB/s    9:57:11

O Comando acima irá comprimir, continuar e salvar parcialmente apresentando a saída em detalhes.

Caso queira usar com ssh:

# rsync -vrlPtz --append -e ssh login@host:/tmp/web.xva /tmp/web.xva
